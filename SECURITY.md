# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this plugin or the Granola MCP server it connects to, please report it responsibly.

**Email**: security@granola.ai

Please include:

- A description of the vulnerability
- Steps to reproduce or a proof of concept
- The potential impact

We will acknowledge receipt within 2 business days and aim to provide a fix or mitigation plan within 7 business days for critical issues.

## Scope

This policy covers:

- The `granola-cursor-plugin` repository (agent instructions, rules, commands, skills, MCP configuration)
- The Granola MCP server at `mcp.granola.ai`

## Out of Scope

- The Granola desktop or web application (report these separately at security@granola.ai)
- Third-party dependencies or services
